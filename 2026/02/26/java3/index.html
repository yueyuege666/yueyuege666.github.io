Java 调用豆瓣 API 获取热门电影实战

发表于2026-02-26|更新于2026-02-26|Java
|浏览量:

任务说明

完成豆瓣热门电影 API 的调用任务，目标 API 地址如下：
https://movie.douban.com/j/search_subjects?type=movie&tag=热门&page_limit=5&page_start=0
需要完成的需求：

  1. 创建 com.youkeda.test.http.ApiAsker 类，调用上述 API；
  2. 先在控制台打印标题 电影名称:；
  3. 在控制台打印前 5 部电影的名字（对应返回数据中的 title 字段）。

问题：

在实际开发中会发现，直接用 Java 程序发起默认请求，豆瓣 API 会拒绝访问，无法获取到正确的电影数据。因为绝大多数正规网站，出于反爬、安全校验、兼容性适配等因素考虑，会对 HTTP 请求进行严格校验，其中最核心的校验项，就是 HTTP 请求头中的 User-Agent 信息。

需要用到的知识点

  1. HTTP Header
    HTTP Header 是 HTTP 协议的重要组成部分，用于在请求中附加 URL 之外的隐藏信息.
  2. User-Agent
    User-Agent 是 Header 中的核心字段，用于告诉服务器当前请求的浏览器类型、版本、操作系统等信息，是服务器校验请求合法性的重要依据。
  3. OkHttp3 配置 Header：
  1. JSON 数据解析
  2. 本次使用阿里的 fastjson 库完成解析，它可以快速将 JSON 字符串转换为 Java 的 Map、List 集合；
  3. 本次 API 返回的 JSON 结构中，电影列表数据存放在 subjects 数组中，数组中的每一个对象，都有一个 title 字段，对应电影的名称。
