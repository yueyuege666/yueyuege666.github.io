快速排序快速排序是基于分治思想的经典排序算法，是编程语言自带排序函数使用最多的算法，快排的核心是选轴分区。 选一个元素作为轴，把数组拆成两部分：比轴小的放左边，比轴大的放右边。 重复以上操作，分别在左右两边再选取元素进行划分轴，划分区。 直到区域内元素的个数为1，则排序完成。 具体操作.利用数组[1, 6, 4, 2, 5, 3]举例，选3作为轴，初始化两头指针：左指针：向右找第一个大于等于轴的元素，右指针：向左找第一个小于等于轴的元素。 左指针找到6停下，右指针找到2停下，交换两个元素，数组变成 [1, 2, 4, 6, 5, 3]。 继续移动指针，左指针找到4停下，右指针也左移到4的位置，两个指针重合，停止移动。 把指针重合位置的4和轴3交换，数组变成 [1, 2, 3, 6, 5, 4]，此时3已经到了它排序后的最终正确位置。 数组被3分成了左右两个子数组[1,2] 和 [6,5,4]，对这两个子数组重复以上的步骤，直到所有子数组有序，最终得到排序结果 [1,2,3,4,5,6]。 代码实现12345678910111213141516171819202122232...
